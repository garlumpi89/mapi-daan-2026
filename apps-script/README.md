# Google Apps Script for RSVP

Copy `Code.gs` into the Apps Script project attached to your Google Sheet and deploy it as a Web App.

The script creates/uses three tabs:

1. `Dashboard` — summary totals and automatic lists.
2. `Mapi&Daan RSVP` — one row per submitted form.
3. `Guest Details` — one row per guest, with menu and allergies/intolerances.

The web currently posts to:
